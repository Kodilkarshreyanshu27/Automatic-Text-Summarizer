# Automatic-Text-Summarizer
Automatic Text Summarizer with various approaches in NLP and Django

### Approahces used 
1. Text Summarization using Gensim with TextRank
2. Text Summarization with Sumy
3. LexRank
4. LSA (Latent semantic analysis)
5. Luhn
6. KL-Sum
7. Summarization with T5 Transformers
8. Summarization with BART Transformers
9. Summarization with GPT-2 Transformers
10. Summarization with XLM Transformers

### Technologies 
1. Django Framework
2. NLTK library for NLP
